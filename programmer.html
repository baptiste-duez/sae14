Niveau 1 — S’intégrer dans un service informatique

Ce niveau regroupe les compétences essentielles pour comprendre un environnement informatique, manipuler des outils de développement, collaborer efficacement et participer à la création d’applications.

AC13.01 — Utiliser un système informatique et ses outils

Découverte et utilisation des outils essentiels : systèmes d’exploitation, gestion de fichiers, outils de développement, terminaux, environnements de travail et bonnes pratiques d’organisation.

AC13.02 — Lire, exécuter, corriger et modifier un programme

Comprendre le fonctionnement d’un programme existant, identifier les erreurs, les corriger et modifier le code pour améliorer ou adapter les fonctionnalités.

AC13.03 — Traduire un algorithme dans un langage donné

Passer d’un algorithme (pseudo-code, organigramme) à un programme fonctionnel dans un langage de programmation adapté (Python, JavaScript, etc.).

AC13.04 — Connaître l’architecture et les technologies d’un site Web

Comprendre le fonctionnement d’un site Web : HTML, CSS, JavaScript, serveurs, bases de données, API, hébergement et interactions client-serveur.

AC13.05 — Choisir les mécanismes de gestion de données adaptés

Identifier les besoins en stockage, sélectionner les technologies adaptées (SQL, NoSQL, fichiers), structurer les données et justifier les choix techniques.

AC13.06 — S’intégrer dans un environnement propice au développement

Travailler efficacement en équipe : versionning (Git), documentation, communication, méthodes agiles, organisation du travail et bonnes pratiques collaboratives.
